Creative Eurorack Patch Ideas with Erica Synths Black Stereo Reverb

The Erica Synths Black Stereo Reverb is a powerful and flexible stereo reverb suited for classic and experimental effects in Eurorack. Here are several creative patch ideas, combining this module with others for unique sonic results:

1. Modulated Ambient Drones

  • Modules to Add: Slow, evolving LFOs (e.g., Mutable Instruments Tides, Xaoc Batumi), random voltage sources (e.g., Make Noise Wogglebug), texture sources (e.g., Rings, Plaits).
  • Patch Idea:
  • Patch a melodic drone source into L (Mono) In.
  • Use several LFOs or random CV sources to modulate Size CV, Tone CV, and Feedback CV for ever-changing reflections.
  • Stereo out to your mixer for lush, ambient stereo fields.
  • Try FREEZE with 1V/oct tracking to create pitch-shifted grains by sending stepped CVs.
  • Tip: Cross-patch LFOs for complex, evolving modulation.

2. Rhythmic Gate Effects

  • Modules to Add: Sequencers/gate generators (e.g., Pamela’s New Workout, 4MS QCD), drum modules (e.g., WMD Crater), clock dividers.
  • Patch Idea:
  • Use rhythmic gates into the FREEZE gate input to stutter reverb tails in sync with your sequence.
  • Use modulatable gate sequences to create ratcheting, shifting rhythmic reverb artifacts.
  • Modulate room size or feedback in sync with the gates for dynamic texture changes.

3. Morphing Reverb Scenes via Preset CV

  • Modules to Add: CV sequencers (e.g., Make Noise Pressure Points, Malekko Voltage Block), addressable switch modules, random voltage sources.
  • Patch Idea:
  • Prepare several presets with dramatically different room types/types/sizes.
  • Use PATCH CV input to morph between presets with a sequencer or random voltage to jump-cut through spaces mid-performance.
  • Map the preset changes to song sections or use for glitchy, non-linear spaces.

4. Feedback & Karplus-Strong Synthesis

  • Modules to Add: External mixer, VCA, audio rate CV sources.
  • Patch Idea:
  • Feed the module’s stereo output back into its input (with proper attenuation via VCA/external mixer).
  • Use short room types (Tape or BBD, Room mode) with high feedback for plucked-string resonator sounds.
  • Modulate feedback and size for dynamic, evolving string-like tones.
  • Ping the system with short envelopes or triggers (e.g., Mutable Peaks, Maths).

5. Stereo Field Automation (SPN/Tone)

  • Modules to Add: Panning/envelope modules (e.g., Intellijel Quadrax, Zadar for complex envelopes), stereo sound sources.
  • Patch Idea:
  • Modulate the Tone/SPN CV to move the reverb’s reflections in the stereo field dynamically.
  • Use envelopes shaped by your other patch elements to send stereo spins on hits or transitions, adding movement and width.
  • Pair with stereo sound sources for maximal spatialization.

6. Performance Effects: Real-Time Freezing

  • Modules to Add: Manual trigger such as a footswitch, tactile gate/trigger modules (e.g., Make Noise Tempi, Mutable Instruments Ears), CV switches.
  • Patch Idea:
  • Control FREEZE with tactile triggers for live loop-style performance.
  • Manipulate SIZE or pitch of the frozen buffer while holding, using a pressure or position controller for expressive granular effects.

7. CV-Controlled Dub Reverb

  • Modules to Add: External delay modules (e.g., Erica Synths Black Delay, Make Noise Mimeophon), hands-on controllers (e.g., Faderbank, Planar2 joystick).
  • Patch Idea:
  • Patch drum or vocal samples through the Black Reverb, then into a delay for extended dub tails.
  • Ride size, feedback, and tone parameters with manual controllers for live dub-style effect throws.

8. Dark Sound Design with "Dirty BBD"

  • Modules to Add: Bitcrusher, distortion, wavefolder, noise generator.
  • Patch Idea:
  • Use the "Dirty BBD" algorithm for degraded/lo-fi spatial textures.
  • Further process the output with bitcrushing for gritty, industrial ambiences.
  • Drive the input with a noisy or distorted source for destructive echo chaos.

General Tips: - The module’s analog dry/wet allows for parallel FX processing; use external VCA or matrix mixers to blend with other effects. - Save presets for different live sets, morphing scenes, or MIDI/CV automation. - Spin parameter can drastically increase perceived space—CV it with random/chaotic sources for trippy results.
